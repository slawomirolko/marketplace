# Olko Kotlin Style Overview

Checks Kotlin/Android style for changed `.kt`/`.kts` files with marketplace defaults: naming, formatting, imports, classes, value classes, extension/scope functions, collection ops, nullability, coroutines/Flow, Compose, and networking style. Runs documented format/lint commands (ktlint/detekt) when configured. Use before commit/test gates or when delegated by olko-commit-style.

Load `SKILL.md` for the canonical workflow and rule list.
