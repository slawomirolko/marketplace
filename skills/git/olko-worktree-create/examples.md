# Smart Worktree Create Examples

## No Plan Context Prompt

```text
No plan in context. What branch name should the worktree use? (e.g. `feature/add-balance-reversal`, `fix/saga-timeout`, `issue/42-empty-results`)
```

## Session Topic Confirmation

```text
Proposed branch name: `feature/add-balance-reversal`. Use this? (y/n)
```
